# ✨ Prompt Buddy!

Translate your rough Roman Hindi (Hinglish) developer thoughts into supercharged, highly structured, expert-level AI prompts instantly!

Built with **React, Vite**, and the **Google Gemini API**, Prompt Buddy bridges the gap between chaotic human thoughts and perfect AI instruction sets using a beautifully animated, dark glassmorphism UI.

## 🚀 Features
- **Hinglish to English Prompt Translation:** Converts broken/roman hindi text into perfectly structured English prompts.
- **Multiple AI Tones:** Choose between Standard Developer, UI/UX Focus, Bug Fixing, Feature Building, Innovative Thinking, and Normal Talk.
- **Dynamic UI:** A stunning layout featuring CSS-animated floating glowing orbs, interactive glassmorphism, and a neon sci-fi feel.
- **Instant Copy:** One-click copy straight to your clipboard for Antigravity, GitHub Copilot, or ChatGPT.
- **100% Private:** Direct execution on your machine via the `.env` API key. Your data never bounces through random servers, straight to Gemini.

## 💻 Tech Stack
- Frontend: React + Vite
- Styling: Plain Modular CSS (Dark, glassmorphic layout)
- Font: *Plus Jakarta Sans* & *Caveat*
- API Integration: Native JS Fetch to Google Gemini `gemini-pro`.

## ⚙️ Setup and Installation
1. Clone the repository
2. Run `npm install`
3. Rename `.env.example` to `.env` and paste your Google Gemini API key:
   `VITE_GEMINI_API_KEY="your_api_key_here"`
4. Run `npm run dev` and navigate to `http://localhost:5173/`
